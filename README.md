# ESP32-C3-MH-Z19C
Показометр CO2 с MQTT и плюшками

CO₂ монитор с датчиком MH-Z19C – Version 2.2

   Что может:
   - Web конфигуратор (Wi-Fi, MQTT, NTP, расписание экрана, кнопка, пороги CO₂, шрифты (в работе), температура)
   - OLED SSD1306 с анимациец для уменьшения статичного выгорания (шарики, плавающие цифры)
   - Два режима кнопки: короткое нажатие (включение на указанное время) / удержание (включение пока удерживается)
   - MQTT публикация CO₂ и температуры (DS18B20)
   - Захардкоженый пин для сохранения настроек в веб-интерфейсе. По умолчанию 0000
   - Симпатичный интерфейс
